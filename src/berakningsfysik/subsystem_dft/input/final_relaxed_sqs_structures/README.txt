# Slutliga relaxerade SQS-strukturer

Här ligger de relaxerade TiAlN SQS-strukturerna som användes för de slutliga SCF-beräkningarna och Delta H_mix.

- ecutwfc = 60 Ry
- ecutrho = 240 Ry
- K_POINTS = 2 2 2
- nstep = 8

Dessa strukturer användes till SCF-energierna som gav den slutliga blandningsentalpin.
